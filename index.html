Hands-on work

Penetration testing home lab — Kali + Metasploitable + Splunk
Built a fully isolated two-VM penetration testing environment on VirtualBox using Kali Linux 2026.1 as the attacker machine and Metasploitable2 as the intentionally vulnerable target. Configured a host-only network (192.168.56.x) to isolate the lab from the home network, added a NAT adapter for Kali internet access, and resolved a persistent NetworkManager/dhcpcd boot-time conflict by writing a custom systemd service. Installed and configured Splunk Enterprise 10.4, exported systemd journal logs to a flat file, and configured Splunk file monitoring to ingest real-time authentication and sudo activity. Completed the first full Nmap service version scan against Metasploitable.
